# banco-de-dados-musica-sql
📌 Sobre o projeto

Projeto desenvolvido para praticar modelagem de banco de dados relacional e linguagem SQL.

🚀 Recursos implementados
Criação de tabelas
Relacionamentos entre tabelas
Chaves primárias
Chaves estrangeiras
Restrições de integridade
Sequências
🛠️ Tecnologias utilizadas
SQL
📚 Conceitos praticados
Modelagem de banco de dados
Relacionamentos
Integridade referencial
Organização de dados
🎯 Objetivo

Aplicar os conceitos de banco de dados relacionais por meio da criação de um banco para uma plataforma de música.
